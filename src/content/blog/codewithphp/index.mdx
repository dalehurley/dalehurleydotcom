## Why I Built Code with PHP

I've spent years working in PHP and teaching others how to use it well. One thing I noticed consistently: most learning resources focus on syntax in isolation — you learn what `foreach` does, but never build anything meaningful with it.

That's the gap [codewithphp.com](https://codewithphp.com) is designed to fill.

The guiding philosophy is simple: **build confidence, not just syntax knowledge**. Every concept on the site is taught through the lens of a real project, so you always understand *why* something matters, not just *what* it does.

## What You'll Find on the Site

### A 30-Day Learning Roadmap

The site is structured around a 30-day roadmap that takes you from PHP fundamentals through to advanced, production-ready patterns. Whether you're brand new to PHP or transitioning from another language, there's a clear path forward.

The roadmap is broken into milestones:

- **Days 1–10** — PHP 8.4 fundamentals, Composer, and project structure
- **Days 11–20** — Routing, MVC architecture, and building a CRM application
- **Days 21–30** — Testing with PHPUnit and Pest, logging with Monolog, and AI/ML integration

### Project-Driven Learning

Rather than covering features in abstract, the site walks you through building complete systems. By the end you'll have built:

- A custom router from scratch
- A full MVC framework
- A CRM application using Laravel 12
- A test suite with PHPUnit and Pest

Each project reinforces the concepts from the surrounding lessons, so nothing feels disconnected.

### Technology Coverage

The site covers the full modern PHP ecosystem:

- **PHP 8.4** — including new syntax features and performance improvements
- **Laravel 12** — the leading PHP framework
- **Composer** — dependency management and autoloading
- **PHPUnit & Pest** — testing your applications with confidence
- **Guzzle & Symfony HttpFoundation** — HTTP clients and request handling
- **Monolog** — professional-grade application logging

## Open Source and Community Driven

The entire site is open source, hosted on GitHub at [dalehurley/codewithphp](https://github.com/dalehurley/codewithphp). If you spot an error, want to suggest a topic, or want to contribute a lesson, pull requests and issues are welcome.

This matters to me. PHP has an incredible community, and a learning resource should reflect that.

## Who Is It For?

Code with PHP is aimed at:

- **Beginners** who want to learn PHP the right way from day one
- **Developers from other languages** (JavaScript, Python, Ruby) who want a structured PHP transition
- **Intermediate PHP developers** who want to sharpen their skills with modern PHP 8.4 patterns and Laravel 12

If you've ever felt like tutorials leave you knowing syntax but not knowing how to *build* something — this site is for you.
